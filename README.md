Welcome! This is my personal portfolio website. As a developer, I have designed this space to showcase my technical skills, creative vision, and the professional standards I bring to every project.

## 💻 Design Philosophy (IDE-Inspired UI)
This portfolio is not your average web page; it is a unique, **IDE (Integrated Development Environment) inspired** project. I wanted to create an environment that reflects the day-to-day workspace of a developer—clean, minimal, and highly functional. By blending modern UI aesthetics with a developer-centric interface, I’ve crafted a platform that is both professional and visually distinct.

## 🤖 The Development Journey
This project was built with the assistance of AI, but it is far from a "copy-paste" job. My development process involved:
* **Customization:** Every design element and logic flow was meticulously tailored to my vision.
* **Rigorous Debugging & Testing:** I treated AI as a collaborative tool, not a shortcut. I spent significant time debugging, testing, and manually refining the code to ensure it is clean, efficient, and "pixel-perfect."
* **Developer Mindset:** My priority was the logic, structure, and performance of the application, ensuring that the code is as high-quality as the visual interface.
